# Trello Login Test Ideas

## Functional
Verify user can login with valid credentials
Verify user cannot login with invalid credentials
Check forgot password flow
verify login with passkey/third party credentials
Check create account flow

## Validation

Login with empty fields
Verify whether email addresses are treated as case-insensitive
Verify that input is capped for username and password
Verify copy/paste credentials


## Security
verify password is hidden

## Usability
Verify clicking outside the login form does not submit the form or cause unexpected behavior
Verify that pressing enter performs login

## Performance
Network interruption during login procedure
Slow network
Click login multiple times

## Session Management
Does logout invalidate the session
Reload page after login
Verify remember me works
